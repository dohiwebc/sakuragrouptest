重要・CONFIDENTIAL

このURL・サイトはチーム関係者のみが利用する内部用です。第三者・他チーム・一般の方への共有・転送・SNS掲載は禁止です。漏えいが起きた場合は速やかに監督・スタッフへ報告してください。

ACCESS アクセス

利用するモードを選択してください。閲覧モードは成績確認、編集モードは記録の入力・更新ができます。

  • 閲覧モード: 成績の閲覧・絞り込み・CSV出力ができます。
  • 編集モード: 大会作成、試合記録、選手管理ができます。
  • 操作は大会選択 → 試合選択 → 選手記録の順で進みます。
  • 保存状態は「記録保存」付近のステータスで確認できます。
  • 出場数 = 選択した試合のうちベンチ以外で出場した試合数（表示は「出場/対象試合」）
  • 集計サマリーの「選択試合数」= 現在の絞り込みで集計している試合数
  • 総アタック数 = アタック成功数 + アタック失敗数
  • アタック成功数 = 合計
  • アタック失敗数 = 合計（ヘッドアタック、ワンバン含む）
  • アタック成功率 = アタック成功数 / 総アタック数
  • 飛んできた本数 = キャッチ数 + アウト数
  • キャッチ数 = 合計（アシストをした数も含む）
  • アウト数 = 合計
  • キャッチ率 = キャッチ数 / 飛んできた本数
  • アタック貢献度 = （各試合での 選手アタック成功数 / チーム総アタック成功数）平均
  • キャッチ貢献度 = （各試合での 選手キャッチ数 / チーム総キャッチ数）平均
  • 内野復帰数 = 外野からの復帰数の合計
  • ゼロ抜き数 = 合計（審判が手を下ろした数）
