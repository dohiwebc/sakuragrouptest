重要・CONFIDENTIAL

このURL・サイトはチーム関係者のみが利用する内部用です。第三者・他チーム・一般の方への共有・転送・SNS掲載は禁止です。漏えいが起きた場合は速やかに監督・スタッフへ報告してください。

ACCESS アクセス

利用するモードを選択してください。閲覧モードは成績確認、編集モードは記録の入力・更新ができます。

  • 試合数 = 現在の絞り込みで集計対象に含まれる試合数
  • 出場数 = スタメンで出場した試合数（表示は「出場/参加大会の対象試合」）
    • 途中交代で入った選手は出場数に含まれません
  • 集計サマリーの「選択試合数」= 現在の絞り込みで集計している試合数
  • 総アタック数 = アタック成功数 + アタック失敗数
  • アタック成功数 = 合計
  • アタック失敗数 = 合計（ヘッドアタック、相手ボールになったワンバン含む）
    • ヘッドアタック：相手ボール（取られたと同じ扱い）
    • 相手ボールになったワンバン：相手にワンバンボールを取られたとき
    • ワンバンが相手にあたってコート外に出た場合は自チームボールなので含みません
  • アタック成功率 = アタック成功数 / 総アタック数
  • 飛んできた本数 = キャッチ数 + アウト数
  • キャッチ数 = 合計（アシストをした数も含む）
    • ワンバンはキャッチ数に含みません
    • アシスト：複数人でボール処理した場合、最終的にボールを持っていた人に適用されます
    • 例：AがボールにあたってBがアシストした場合、Bのキャッチとしてカウントされます
  • アウト数 = 合計
  • キャッチ率 = キャッチ数 / 飛んできた本数
  • アタック貢献度 = （各試合での 選手アタック成功数 / チーム総アタック成功数）平均
  • キャッチ貢献度 = （各試合での 選手キャッチ数 / チーム総キャッチ数）平均
  • 内野復帰数 = 外野からの復帰数の合計
  • ゼロ抜き数 = 合計（審判が手を下ろした数）

オンのとき、まだチュートリアルを完了していない人だけがホームで案内を見ます。設定は Firebase の共有データに保存され、サイトを開く全員に適用されます。

オフにすると「?」は薄く表示されたままになり、押すと「使用できません」と案内だけ出ます（自動表示は上段の設定に従います）。
